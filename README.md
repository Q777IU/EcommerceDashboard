# 电商数据可视化平台

基于 Streamlit + Pandas + Plotly 构建的电商数据分析可视化平台，支持多维度数据洞察和交互式图表展示。

## ✨ 功能特性

- **📈 整体概况**：核心指标仪表盘（销售额、利润、订单数、转化率等）
- **💰 销售分析**：品类销售占比、支付方式分布、销售排行
- **👥 用户分析**：会员等级分析、性别/年龄段消费分布、TOP用户排行
- **📦 商品分析**：商品销售排行、品类对比、利润-销量散点图
- **🌍 地区分析**：省份销售额排行、地区占比分析
- **⏰ 时段分析**：24小时销售分布、星期分布、日粒度趋势

## 🛠️ 技术栈

- **前端框架**: Streamlit
- **数据处理**: Pandas + NumPy
- **可视化**: Plotly (交互式图表)
- **数据存储**: Excel / Pickle

## 📦 安装依赖

```bash
pip install -r requirements.txt
```

## 🚀 快速启动

### 方式一：双击启动（推荐）

直接双击 `启动电商数据平台.bat` 文件即可启动。

### 方式二：命令行启动

```bash
# 首次运行需生成数据
python data_generator.py

# 转换数据为pickle格式（加速加载）
python convert_to_pickle.py

# 启动可视化平台
streamlit run app.py --server.port 8501
```

启动后访问 `http://localhost:8501` 即可使用。
